## What Is Migratowl?

Migratowl is an AI-powered dependency migration analyzer. It answers one question: **"If I upgrade this dependency, will anything break — and how do I fix it?"**

For full documentation, architecture details, API reference, and setup instructions, see the [new repository](https://github.com/bitkaio/migratowl).
